# Swipe File

Curated examples of high-performing content, organized by platform.

## Directories

- **`linkedin/`** — LinkedIn posts: hooks, structure, engagement patterns
- **`blog/`** — Blog articles: SEO structure, section rhythm, intro hooks

## Usage

The writer skills reference their respective swipe directories automatically:
- `blog-writer` reads from `blog/`
- `linkedin-writer` reads from `linkedin/`

Learn patterns — never copy content.
