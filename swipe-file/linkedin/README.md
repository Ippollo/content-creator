# Swipe File

Your curated collection of LinkedIn posts you admire — both your own top performers and others' posts.

## How to Use

1. **Save a post**: Create a new `.md` file in this folder (e.g., `01-bold-opinion.md`)
2. **Follow the template**: Use the format in `_template.md`
3. **Tag it**: Note the archetype and what you like about it
4. **The agent does the rest**: When writing posts, the agent reads these examples and learns structural patterns

## Tips

- You don't need dozens. **5–10 great examples** is plenty to start.
- Mix archetypes — hooks, stories, frameworks, opinions
- Include at least a few of your own posts that performed well
- Note _what makes it work_, not just the content
