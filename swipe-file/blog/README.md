# Blog Swipe File

Curated examples of long-form blog posts that demonstrate effective structure, storytelling, and SEO practices.

## How to Use

Save blog posts you admire as markdown files in this directory. The `blog-writer` skill will analyze them for patterns before writing.

## What to Look For

When saving a swipe example, capture:

- **Intro hook**: How does it open? (bold claim, story, statistic, question)
- **Heading structure**: How are H2/H3s organized? (chronological, problem-solution, framework)
- **Section rhythm**: Length of sections, variety of paragraph lengths, use of lists vs prose
- **SEO signals**: How naturally are keywords woven in? Meta description quality?
- **CTA/Conclusion**: How does it close? (summary, call-to-action, open question)

## File Format

Use the same template pattern as the LinkedIn swipe file — one file per example with a brief analysis header.
