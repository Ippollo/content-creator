# X Swipe File

This directory contains examples of high-performing, build-in-public X posts and threads. The `x-writer` skill uses these as structural and tonal references when drafting new content.

## Curation Guidelines

To add a new example:
1. Copy `_template.md` to a new slug (e.g., `2026-04-10-contrarian-hook.md`).
2. Paste the raw text of the post/thread.
3. Fill out the YAML frontmatter, specifically categorizing the `format` and `hook_pattern`.
4. Add 1-2 sentences explaining *why* it worked (e.g., "The hook sets up a high friction problem, the body lists concrete tech stack decisions, no fluff").

*Note: The `x-writer` skill is currently optimized for build-in-public and technical operations niches. Avoid curating abstract philosophy tweets unless they directly map to a business/ops outcome.*
