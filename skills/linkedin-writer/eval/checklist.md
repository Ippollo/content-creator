# LinkedIn Post Checklist

Run each check against the drafted post. A post must pass ALL items before proceeding to content-reviewer scoring.

## Mechanical Checks

These are binary — the post either passes or fails. No judgment call needed.

- [ ] **No em dashes (—)**: Search for `—`. If found, replace with a period, comma, or restructure the sentence.
- [ ] **No hashtags**: Search for `#`. If any hashtag is present, remove it.
- [ ] **No emojis in the first line**: Check the opening line for any emoji characters.
- [ ] **Character count 800–1300**: Count the full post (excluding the CTA question). Flag if outside range.
- [ ] **No external links in post body**: Links suppress reach. If present, move to first-comment.

## Voice Checks

These require reading against the voice profile. Flag any violation.

- [ ] **No corporate-speak**: Scan for phrases like "In today's fast-paced world", "It's important to note", "leverage", "synergies", "actionable insights". If found, rewrite in plain language.
- [ ] **No AI tells**: The post should not sound like it was generated by an AI. Check for overly smooth, hedging, or generic language.
- [ ] **Plain language filter**: Would every word come up in a conversation with a friend? Common swaps: "diagnostic" → "the right", "methodology" → "framework", "optimize" → "improve".
- [ ] **No over-explanation**: If a line makes a point and the next line re-explains it in different words, cut the second line.

## Structural Checks

These verify the post architecture.

- [ ] **Hook stops the scroll**: The first 1-2 lines must create curiosity, tension, or a bold claim. If the hook is a passive observation, rewrite.
- [ ] **Doesn't open with throwaway "I"**: "I think..." or "I've been reflecting..." are weak. "I built..." or "I tracked..." with specific proof is fine.
- [ ] **Single core argument**: Can you remove any section and the post's core argument still stands complete? If yes, that section is a second idea — cut or rework.
- [ ] **CTA present and specific**: The post ends with a direct question or invitation to engage. Not "thoughts?" — something specific to the topic.
- [ ] **All stats have named sources**: If a number or statistic appears, it must be attributable to a named source (e.g., "Microsoft's 2025 Work Trend Index"). No made-up stats.
- [ ] **Cascading lists**: If the post uses a bullet/emoji list, items should be progressively shorter or longer. Rewrite items to achieve the cascade if logical order prevents reordering.
