# X Writer: Mechanical Gate Checklist

Run this checklist against the draft before sending it to the `content-reviewer`. If any item fails, fix it immediately. 

## 1. Algorithmic Violations
- [ ] **No external links in the main text**: If a link is required, it must be in a separate reply tweet.
- [ ] **No hashtag stuffing**: Maximum of 1 targeted hashtag, but 0 is preferred.
- [ ] **No engagement bait**: No "Reply below" or "Retweet to share". 

## 2. Structural & Mechanical Rules
- [ ] **Character Limits Validated**: Every tweet in the draft is ≤ 280 characters.
- [ ] **No Em Dashes (—)**: Replaced with commas, periods, or restructuring.
- [ ] **No Emoji Openers**: The first line (highest leverage hook area) does not start with an emoji.
- [ ] **Visual Rhythm**: Does not look like a uniform wall of text. Uses line breaks effectively.

## 3. Credibility & Voice
- [ ] **Zero Corporate Speak**: Words like "delighted", "thrilled", "unleash", or "innovative" are removed.
- [ ] **Specifics Present**: Includes at least one concrete detail (a metric, a specific tool name, a timespan).
- [ ] **EHA Structure (For Threads/Singles)**: The post uses some form of Emotional Trigger → Hook → Action to construct the narrative.
